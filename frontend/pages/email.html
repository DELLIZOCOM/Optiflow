Before you connect (Azure AD)
  1. In Azure Portal → App registrations, register a new application.
  2. Under API permissions, add Microsoft Graph → Application → Mail.Read and User.Read.All, then click Grant admin consent.
  3. Under Certificates & secrets, create a new client secret and copy the Value.
  4. Paste the Tenant ID, Client ID, and Client Secret below.
GoDaddy Workspace Email
  1. Server is preset to imap.secureserver.net on port 993 (SSL).
  2. Add each company mailbox below using its full email address and password.
  3. If your mailbox uses 2-factor login, generate an app-specific password in GoDaddy and use it here.
Generic IMAP server
  1. Find your provider's IMAP host and port (commonly imap.<domain>.com on 993 with TLS).
  2. Use plain port 143 only on a trusted internal network.
  3. Add one row below per mailbox you want indexed.
